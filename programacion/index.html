Programacion. Contexto. Python
    En ciencia es de utilidad el sistema operativo Linux este es un software de código abierto y útil para grandes sistemas, multi-usuario, multi-tarea y de uso intensivo de la red. En la asignatura uno de los objetivos es aprender la esencia de linux.

    Para la programación científica hay dos grandes tipos de lenguajes los intérpretes (scripting) y los compilables. Del primer grupo se pueden citar a python y R (de código abierto), IDL y Matlab (pagos). Mientras del segundo grupo podemos citar a Fortran y C.

    En la asignatura aprenderemos a programar a través del lenguaje Python .
    Es un lenguaje relativamente reciente que ha crecido enormemente en la última década. Gran parte de las aplicaciones en inteligencia artificial, aprendizaje automatizado se encuentran desarrolladas en python.

    Python fue creado por Guido vanRossum, un programador holandes en 1991. La historia de python puede ser leída aqui
    Python es el lenguaje actualmente mas utilizado para el aprendizaje de programacion. Esto se debe a sus principales características:
    • Fácil e intuitivo.
    • Es de código abierto.
    • Posee una gran cantidad de usuarios, de librerías disponibles y por ende de foros e.g. stackoverflow (sitio web de preguntas y respuestas de lenguajes).

    Otras herramientas que deben aprender a utilizar:
  • Terminal o consola. Lenguaje shell / bash. Manejo de archivos y administrar la computadora en general.
  • Editor de textos (para programar). Nano, Notepad (simples), mas especializados: vi, emacs. Microsoft Word NO sirve.
  • Procesador de textos. Para fórmulas matemáticas se utiliza el LaTeX. Para una introducción ver este material de un taller de latex.
  • Otro procesador de textos mas sencillo que es de utilidad: markdown (Si no hay mas remedio: LibreOffice o Microsoft Word).

    Como ejemplo del software que uso:
    • Sistema operativo: linux (distribución opensuse).
    • Administrador de ventanas: ion, es uno ultrasimple y muy liviano. Para principiantes se recomienda gnome/kde.
    • Terminal: bash, screen.
    • Editor de textos: emacs.
    • Lenguajes principales: fortran y python.
    • Procesador de textos: latex.
    • Herramienta para reproducible research que combina múltiples lenguajes de programación, textos y figuras: org-mode.
    • Para ver archivos pdf: okular. Para ver figuras: gwenview
    • Versionador de software: git. (repositorios: gitlab, github, bitbucket).
    • Navegadores: firefox, chrome, lynx.
    • Herramientas HPC: openmpi, ganglia, torque.
