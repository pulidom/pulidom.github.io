COVID-19 data assimilation and probabilistic prediction in Argentina

Some early results using data assimilation techniques for COVID-19 data processing and prediction.
One of the main advantages of data assimilation techniques is that we can infer a hidden state from partial observations. The data assimilation technique combines epidemiological models and several sources of observations quantifying the uncertainty in the model predictions and the hidden state estimation.
SEIR model combined with EnKF with the augmented state approach.


(a) Observations of the infected number (orange dots) and the estimated infected number with data assimilation (cyan line). Gray lines show the ensemble representing the uncertainty of the estimation. For dates where there are no observations yet, a prediction is conductd with the last set of estimated parameters and started from the last estimated hidden state. First day corresponds to March 3rd.
(b) Reproduction number estimated with the data assimilation technique and its uncertainty.
